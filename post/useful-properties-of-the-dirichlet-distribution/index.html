Useful Properties of the Dirichlet Distribution

Last updated on Jun 7, 2020 2 min read Bayesian Nonparametrics

This post is a continuation of the post that constructs a Dirichlet distribution from Gamma distributed variables. We use the same parameterization of the Gamma Distribution as before, and set the rate parameter \(=1\) unless otherwise specified. In this post I prove that the following property holds:

If \(U = (U_1,\dots,U_{i},U_{i+1}, \dots, U_{N}) \sim Dir(\alpha_1,\dots,\alpha_i,\alpha_{i+1},\dots, \alpha_N)\) then \(U' = (U_1,U_2,\dots,U_{i}+U_{i+1}, \dots, U_{N}) \sim Dir(\alpha_1,\alpha_2,\dots,\alpha_i+\alpha_{i+1},\dots, \alpha_N)\)

In general, \(U'' = (\sum\limits_{i=1}^{k_1} U_i, \sum\limits_{i=k_1+1}^{k_2} U_i, \dots, \sum\limits_{i=k_j+1}^{N} U_i ) \sim Dir(\sum\limits_{i=1}^{k_1} \alpha_i, \sum\limits_{i=k_1+1}^{k_2} \alpha_i, \dots, \sum\limits_{i=k_j+1}^{N} \alpha_i )\)

I prove the first part of this property here. The rest follows by simply extending the proof. The strategy here is to construct IID Gamma distributed variables with appropriate parameters, and then derive the distribution of \(U'\) using the results from the previous post.

Assume \(Z_i\)’s are IID \(\sim Gamma(\alpha_i,1)\). From the additive property of the Gamma distribution, I have \(Z_{i,i+1} = Z_i + Z_{i+1} \sim Gamma(\alpha_i+\alpha_{i+1},1)\). By definition, \(Z_{i,i+1}\) is independent of \(Z_j\)’s for \(1 \le j \le N, j \neq i, i+1\).

I now define \(U' = (U_1,U_2,\dots,U_{i}+U_{i+1}, \dots, U_{N}) = (U_1,U_2,\dots,U_{i,i+1}, \dots, U_{N})\), where \(U_j = \frac{Z_j}{\sum\limits_{j\neq i, i+1}^{N} Z_j + Z_{i,i+1}}\). From here, it follows that \(U' \sim Dir(\alpha_1,\alpha_2,\dots,\alpha_i+\alpha_{i+1},\dots, \alpha_N)\). The second part of theorem follows from similar reasoning.

The result above can be used to get the marginal distributions of the \(U_i\)’s, where \(U = (U_1,\dots,U_i,\dots,U_N)\).

Define \(U' = (U_i, \sum\limits_{j \neq i}^{N} U_j)\).

From the result above, this is distributed \(Dir(\alpha_i, \sum\limits_{j \neq i}^{N} \alpha_j) = Beta(\alpha_i, \sum\limits_{j \neq i}^{N} \alpha_j)\).

Since \(\sum\limits_{j \neq i}^{N} U_j = 1 - U_i\), the density function can be expressed purely as a function of \(U_i\), which is the marginal distribution of \(U_i\).

Formally:

If \(U = (U_1, \dots, U_{N}) \sim Dir(\alpha_1,\dots, \alpha_N)\) then \(U_i \sim Beta(\alpha_i, \sum\limits_{j \neq i}^{N} \alpha_j)\)
